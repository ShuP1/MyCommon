# MyCommon

MyCommon is a collection of usefull Classes
* MultiLang - 10/12/2016
* ConsoleIO - 11/12/2016  <sup>(from MyConsole)<sup/>
* Binary
* Logger
* Strings

### Prerequisities

* .NET/[Mono](https://github.com/mono/mono) 4.5+
